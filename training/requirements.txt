# Versions chosen for Vast.ai instances on the
# pytorch/pytorch:2.4.0-cuda12.1-cudnn9-runtime base image.
#
# Philosophy:
#   - torch is NOT pinned: the Vast.ai image ships a working CUDA-matched build.
#     Touching it can break the CUDA install. We trust the image's torch.
#   - unsloth-zoo is NOT pinned: it's a strict dep of unsloth and the version
#     range moves together. Pinning both fights pip.
#   - Everything else gets a >= lower bound for reproducibility, with an
#     upper bound to block known-breaking majors.
