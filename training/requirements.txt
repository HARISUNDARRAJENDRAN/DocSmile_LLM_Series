# DocSmile CPT training requirements
#
# Target Vast.ai image:
#   pytorch/pytorch:2.4.0-cuda12.1-cudnn9-runtime
#   (pytorch 2.5.x with CUDA 12.x also works)
#
# DO NOT pick a bleeding-edge nightly image (torch >= 2.8 / CUDA 13).
# Unsloth, flash-attn, and bitsandbytes don't have wheels for those yet.
#
# Philosophy:
#   - torch: NOT pinned here. setup_vast.sh adds a runtime constraint so the
#     image's torch is never replaced.
#   - unsloth-zoo: NOT pinned. Strict dep of unsloth; pip resolves it.
#   - transformers: deliberately unpinned. Unsloth pins it itself, and the
#     valid range shifts every minor unsloth release. Let unsloth win.
#   - bitsandbytes: unpinned upper bound — 0.45+ uses different CUDA wheel
#     layout. unsloth tracks compatible versions.
